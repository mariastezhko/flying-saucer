## Description

`Flying Saucer` is an Augmented Reality iOS Application.

Key features of `Flying Saucer` include:

- Fully controllable 3D object that can rotate and move in all directions.
- Sound effects

Technologies used:

iOS, Xcode, Swift, ARKit

UFO 3D model by Herminio Nieves@2013
https://free3d.com/3d-model/ufo-21509.html
